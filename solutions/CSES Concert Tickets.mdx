## Solution

[Unofficial Editorial](https://codeforces.com/blog/entry/83295)

Another way of solving the problem is to use the upper_bound function to find the lowest-priced ticket which exceeds the maximum price for the current customer. When we have found the iterator pointing to the upper bound, just decrement the iterator to get the maximum price possible for that current customer . Note that if the iterator points to the begin of the multiset, we can infer that there is such no ticket having the satisfied price for the customer. ([Proof for my solution](https://stackoverflow.com/questions/9989731/algorithm-function-for-finding-last-item-less-than-or-equal-to-like-lower-bou/9989773#9989773)) 

```cpp
#include <bits/stdc++.h>
 
using namespace std;
 
//variables used for the current problem
int n,m,h,t; multiset<int> tickets;
 
void solve() {
	cin >> n >> m;
	for (int i=0;i<n;++i){
		cin >> h; tickets.insert(h);
	}
	for (int i=0;i<m;++i){
		cin >> t;
		auto it = tickets.upper_bound(t);
		if (it==tickets.begin()){
			cout << -1 << endl;
		}
		else{
			cout << *(--it) << endl;
			tickets.erase(it);
		}
	}	
}

int main(){
	ios::sync_with_stdio(0);
	cin.tie(0);
	solve();
	return 0;
}
```
