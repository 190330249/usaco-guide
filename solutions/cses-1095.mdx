## Explanation

[Fermat's Little Theorem](gold/modular) tells us that $a^{p - 1} \equiv 1 \pmod{p}$, so we can calculate $a^{b^c \pmod{1e9 + 7 - 1}} \pmod{1e9+7}$ with modular exponentiation.

## Implementation

**Time Complexity:** $\mathcal{O}(\log P)$

<CPPSection>

```cpp
#include <bits/stdc++.h>
using namespace std;

const long long MOD = 1e9 + 7;

long long binpow(long long x, long long n, long long m) {
	assert(n >= 0);
	x %= m;
	long long res = 1;
	while (n > 0) {
		if (n % 2 == 1)  // if n is odd
			res = res * x % m;
		x = x * x % m;
		n /= 2;  // divide by two
	}
	return res;
}

void solve() {
	long long a;
	long long b;
	long long c;

	cin >> a >> b >> c;

	long long pow_bc = binpow(b, c, MOD - 1);
	long long ans = binpow(a, pow_bc, MOD);

	cout << ans << '\n';
}

int main() {
	int t;
	cin >> t;

	while (t--) {
		solve();
	}
}
```
